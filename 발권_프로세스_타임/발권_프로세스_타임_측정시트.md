## 📋 측정 가이드

### 측정 방법
1. 스톱워치로 각 작업의 시작/종료 시간 측정
2. 최소 5~10건 측정 (권장: 10건 이상)
3. 특이 케이스(에러, 네트워크 장애 등) 제외
4. 아래 시트에 평균 시간 기록

### 측정 단위
- 시간 단위: **분** (예: 3분, 180분)
- 1분 미만은 초 단위 (예: 30초)

## 💡 측정 팁

### 처리 시간 vs 대기 시간
- **처리 시간**: 실제 작업 수행 시간
- **대기 시간**: 고객 응답, 외부 승인 대기 등
- 대기 시간도 별도 기록 (병목 파악 중요)

### 측정 예시
```
케이스: V001
- 미확정 사유 확인: 09:00 시작 → 09:03 완료 = 3분
- 코멘트 검토: 09:03 시작 → 09:05 완료 = 2분
- TL 연장: 09:05 시작 → 09:10 완료 = 5분
```

### 샘플 수집 기준
- 최소 5건: 기본 평균 파악
- 권장 10건: 신뢰도 있는 평균
- 20건 이상: 최소/최대값 분석 가능

### 대기 시간 측정 주의
- **컨펌 회신**: 이메일 발송 후 고객 응답까지 대기 시간
- **서류 업로드**: 요청 발송 후 고객 업로드까지 대기 시간
- 대기 시간은 보통 매우 길 수 있습니다 (수 시간~하루)
