LAFC 직사입 티켓 운영 가이드

📋 개요

상품

손흥민 로스앤젤레스 풋볼 클럽(LAFC) 직사입 티켓

운영 방식

  • B2B: 파트너사에게 벌크 발권 → 파트너사가 고객에게 배부

  • B2C: 마리트가 고객에게 직접 발권

🔑 기본 정책

정책

내용

❌ 환불

확정 후 환불 불가

🪑 자리 배정

발권 순서대로 자동 배정 (예외 요청/배정 후 변경 불가)

한 주문 당 수량 내에서는 연속된 좌석으로 배정

(1장씩 따로 구매 시 연석 배정 불가)

✅ 양도

고객끼리 양도 가능

📝 발권 프로세스

1. 시트에서 예약 내역 확인

  • 고객 정보, 구매옵션, 예약 수량 확인

2. 마리트 확정 처리

  • 확정대기 → 확정

  • 확정을 먼저 눌러주는 이유

  • 확정대기 상태에서 여행자 취소해버리면 우리가 수동 발권하는 도중일 수 있으니 먼저 확정을 함

    • 티켓 마스터에 존재하는 재고만큼 상품에 재고 업로드 됨

    • 다만 티켓마스터 사이트 접속 시 VPN OFF 가 필요하여 매니저 페이지 확정 후 처리

    • B2C 배정 시 연석 지정 불가한 경우

      • 확정대기 상태로 두고

      • 여행자 확정까지 최대 ~1일 소요 안내

      • 재고 이동 관련하여 사업실 조유진,김정환,노경현,이유진 님 문의

      • 확정 또는 취소 처리

3. LAFC 사이트 발권

  • 발권 사이트 접속 → 로그인 → 발권 실행

    • 발권 실행 전 발권 시트 재고 트래킹 영역에서 B2B, B2C 티켓 좌석 확인 필요 상태값 정보 기입

  • 티켓 전달

4. 좌석 정보 기입

  • 시트에 기록: 티켓번호, Section, Row, Seat, 발권 일시, 발송여부, 확정여부 기입

상태 흐름: 예약접수 → 확정대기 → 확정완료 → 발권진행중 → 발권완료 → 티켓전달완료
