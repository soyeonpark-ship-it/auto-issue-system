⭕ 상품 등록 방법

순서

상세 화면

상세 내용

1

매니저 > T&A 연동 상품

  • [T&A 연동 상품] 버튼을 눌러주세요

2

매니저 > T&A 연동 상품

  • 검수가 필요한 상품의 공급사를 클릭해주세요

3

공급사 상품 가져오기

  • [공급사 상품 가져오기] 버튼을 눌러주세요

  • 이미 등록된 상품을 중복으로 등록할 경우, 아와 같은 에러 문구가 노출됩니다

  • 공급사 ID 를 기준으로 검색한 후, 중복 상품이 없다면 등록을 시작해주세요

4

공급사 상품 가져오기

  • 등록하고 싶은 공급사의 원본 ID를 넣어주세요

    • 1개만 가져오요

    • 여러개 가져오기를 할 경우 , 로 구분값을 넣고 등록하면 됩니다

예시) 온세일 시트 내 “Product ID” 열에 기재된 공급사 페이지 링크에서 상품 번호 입력

5

공급사 상품 가져오기

  • 상품 등록 완료 문구가 노출되었다면, 연동 상품 ID 를 검색해주세요

6

상품 생성하기

  • [마리트 상품 생성] 버튼을 눌러주세요

7

표준 카테고리 선택

  • 표준 카테고리는 필수 값이므로 선택 후 마리트 상품을 생성할 수 있습니다

  • 표준 카테고리 선택 후 [연동 상품 등록하기] 버튼을 클릭하면 마리트 상품 생성이 완료됩니다

8

상품 상세 정보 입력하기

  • 마리트 상품 ID 를 클릭하면 파트너 페이지로 이동합니다

  • 각 페이지에서 필수 값을 입력한 후 [심사 요청] 버튼을 누르세요

    • 대표 도시

    • 상품명

    • 상품 사진

      • 최소 4장 (~9/1 까지)

      • 최소 1장 (9/1부터~)

    • 상품 소개

      • 별도로 템플릿은 없고 공급사 페이지 내 내용을 거의 복사 붙여넣기 해주시면 됩니다

      • 만약!! 온세일 시트에서 “상세 내용 이미지” 열에 체크해주신 건이라면 사업실에서 이미지를 다시 넣으실 건이라 상품 소개는 아주 간략하게만 넣어두시다

    • 이동 수단 (투어)

    • 총 소요시간 (투어)

    • 사용 언어 (투어)

    • 공지사항/이벤트

    • 만나는 시간

    • 예약 마감 시간

    • 만나는 장소

    • 포함 사항

    • 불포함 사항

    • 필수 안내 사항

8-1

헷갈리는 내용 꼭 참고!!

  • 처음 상품 정보 등록 시에는 임시저장 / 심사요청 버튼만 노출됩니다

    • 모든 정보 입력 후 [심사요청] 해도 심사요청 상태로 바뀌지 않을 수 있고

    • 새로고침하시면 버튼명이 바뀔거에요

← 이렇게 버튼명이 2개 노출될 때에 (1) 상품 정보 수정 (2) 심사요청 버튼을 순차적으로 눌러주셔야해요!!

  • 이렇게 완료되면 매니저 페이지에서 “심사중” 상태로 바뀐 점을 확인할 수 있습니다

9

3.0 매니저페이지 최종 설정

  • 예약정보

    • 예약 가능 시간

      • 즉시 확정이라면 0 으로 설정

    • 예약 확정 방식

    • 바로 예약

      • “바로 예약”

    • 결제수단 나중결제 가능 여부

      • “사용 불가”

    • 예약 시 본인 인증 필요여부

      • “불필요”

    • 현금 영수증 발행 여부

      • “미발행”

  • 취소/환불 정보 수정

    • 취소/환불 템플릿

      • “미사용”

        • 공급사와 동일한 취소환불 정책으로 운영하기 때문에 취소환불템플릿을 사용하지 않아야합니다

    • 취소/환불 문구 입력하기

      • 상품에 특별한 정책이 없는 경우, 아래와 같은 디폴트 문구 작성을 해주세요

        • 옵션 별로 취소/환불 정책이 상이할 수 있습니다. 옵션 선택 후 상세 정책을 확인해주세요.

    • 취소 요청 가능 여부

      • “취소 불가”

        • 확정 후 환불 불가 규정

      • “취소 가능”

        • 확정 후 환불 가능 규정

  • 대표가

    • (KKDAY에 한해서만) 사업실에서 설정해주기로 하였으므로 온세일 시트 L열에 “상품조성완료 -> 대표가 등록필요” 로 기재하고 슬랙 스레드에 노티해주세요

    • 클룩 신규 상품 등록 시 판매가 정책 셋팅 방법
      : 신규 상품의 경우 따로 판매가정책 등록 없이 판매시작!
      → 상품 조성 완료 후 바로 심사 요청 > 판매 시작 !
