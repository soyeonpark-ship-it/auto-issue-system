엘에스 컴퍼니 상품 등록 방법

1) 기존 딜이 있는 경우 : 맵핑 변경

순서

상세 화면

상세 내용

1

공급사 슬랙 요청 확인
※ 맵핑 변경은 기존 딜이 있는 모든 공급사의 상품 페이지 등록/ 원본 페이지 변경 등에 공통 적용 가능합니다.

2

MRT ID 로 검색하여 2.0 매니저 페이지 진입

3

판매 정지 (REJECT) 처리 후 생텀으로 돌아와 해제 버튼 클릭
※ 2.0 페이지 (MRT ID : 114571) 과 기존 원본 ID (원본 ID : PR25053053) 을 분리하는 작업

4

새로 전달 받은 원본 ID 를 생텀으로 불러오기
→ 우측 하단 초록색 가방 모양 클릭 (상품 가져오기) 후 가져올 원본 ID 입력
→ 가져오기 클릭 하여 신규로 불러오기

5

신규로 불러온 ID 에 기존에서 떼어넨 MRT ID (114571) 입력 후 변경 클릭
→ 변경 완료 된 후 [전체 적용]을 눌러 새로 불러온 공급사 상품 내역이 기존 2.0 페이지에 반영 되도록 한다

6

2.0 페이지로 들어와 도시 선택 > 정보수정 > 기본 정보 내 수정 필요 내역 있다면 수정 후 [판매시작]

7

상품 온세일 후 수수료 등록을 위하여 “[국내T&A] 맵핑변경, 신규 상품 등록” 시트 내 맵핑변경 내역 기재 (A-F열)




2) 신규 등록의 경우

순서

상세 화면

상세 내용

1

공급사 슬랙 요청 확인

2

기존 딜이 있는지 확인
→ 생텀 공급사 메뉴에서 [상품 제목] 으로 검색
→ 기존 딜이 있다면 맵핑 변경으로 진행 / 없다면 신규 등록 진행

3

원본 ID 불러오기

4

MRT ID 생성되면 [전체 적용] 클릭하여 옵션/이미지 등 전체 내용 2.0 적용

5

2.0 매니저 페이지 등록
→ 도시 선택 / 정보 수정 등록


※ 정보 수정 등록 시 필수 확인 사항
→ 알림 방법 : 예약 후 카카오톡 혹은 SMS 로 티켓 발송 되는 상품의 경우, “ SMS 또는 카카오톡 전송상품용 템플릿” 선택 필수
→노출 옵션
▷ 유효기간 내 이용하는 상품 중 최종 이용 가능일만 있는 경우,
▶ “상품 상세 페이지에서 옵션 노출” 체크
▶ 상품 만료 기한 기재 필수
▷ 유효기간 내 이용 하는 상품 중 30/90/100일 이내 이용 가능한 상품 / 최종 이용 가능일은 기한이 없는 경우,
▶ “상품 상세 페이지에서 옵션 노출” 체크
▶ 상품 만료 기한 : 2099- 12-31 로 기재
▶ 구매일로 부터 N 일 : 유효한 기간 (30/60/90/365 등) 기재


6

상품 상세페이지 수정
→ 상품 이름을 한줄 요약 및 여행 소개에 동일하게 기재 (필수 값으로 기재하지 않으면 온세일 불가)

7

모두 수정 후 [판매 시작]

8

상품 온세일 후 수수료 등록을 위하여 “[국내T&A] 맵핑변경, 신규 상품 등록” 시트 내 신규 등록 내역 기재 (A-F열)

순서

상세 화면

상세 내용

1

공급사 슬랙 요청 확인
→ 전달 받아야 하는 정보가 모두 있는지 확인 필요
- 상품명
- 판매 기간
- 이용 기간 / 유효기간
- 수수료
- 옵션 상세 내역 : 옵션 코드, 옵션명 , 옵션 별 판매 가격
- 썸네일 이미지 (여행 사진)
- 딜페이지 이미지 (상세 이미지)

2

공급사 신규 상품 등록
→ 기존 딜 있는지 확인 후, 없다면 신규 등록 진행 (기존 딜 있다면 맵핑 변경 - 상단 엘에스 컴퍼니 맵핑 변경 참고)
→ 신규 상품 생성 > 상품명 기재 후 등록 누르면 생텀 페이지 생성됨

3

MRT ID 생성

4

옵션 정보 등록
→ 옵션 추가 클릭 하면 등록 할 수 있는 옵션 란이 늘어남
→ 옵션명/설명/연동코드/권장가/수수료/판매가/시작일/종료일 등록 (공급사에서 전달해준 스레드 내용대로 기재)

※ 옵션 정보 기재 시 주의 점
→ 상품 연동 코드가 올바르게 기재 되어있는지 확인
→ 권장가와 판매가가 다른 경우 > 판매가로 권장가/판매가 모두 기재
→ 설명 : 유효기간이나 나이 기준 등 기재
→ 시작일/종료일 : 판매 기간 기재 > 옵션이 노출 되는 시작과 종료일을 뜻함

5

옵션 정보 모두 기재 후, [수정 사항 반영] 클릭

※ 반영하지 않으면 기재한 옵션 내역 초기화됨

6

2.0 매니저 페이지 기본 정보 등록
→ 도시 선택 / 정보 수정 등록
→ [수정] 누르고 들어가 상세 페이지 수정 시작




※ 정보 수정 등록 시 필수 확인 사항
→ 알림 방법 : 예약 후 카카오톡 혹은 SMS 로 티켓 발송 되는 상품의 경우, “ SMS 또는 카카오톡 전송상품용 템플릿” 선택 필수
→노출 옵션
▷ 유효기간 내 이용하는 상품 중 최종 이용 가능일만 있는 경우,
▶ “상품 상세 페이지에서 옵션 노출” 체크
▶ 상품 만료 기한 기재 필수
▷ 유효기간 내 이용 하는 상품 중 30/90/100일 이내 이용 가능한 상품 / 최종 이용 가능일은 기한이 없는 경우,
▶ “상품 상세 페이지에서 옵션 노출” 체크
▶ 상품 만료 기한 : 2099- 12-31 로 기재
▶ 구매일로 부터 N 일 : 유효한 기간 (30/60/90/365 등) 기재

7

상품 상세 페이지 등록
→ 상품 이름을 한줄 요약 및 여행 소개에 동일하게 기재 (필수 값으로 기재하지 않으면 온세일 불가)
→ 여행 사진 업로드 : 썸네일 이미지 등록

8

상세 이미지 등록
→ 2.0 매니저 메인 화면에서 상세 이미지 관리 클릭
→ 새 이미지 업로드 > 이미지 선택 후 (중복 선택 가능) 확인 > 업로드 하기 클릭

9

업로드 된 이미지 저장
→ 업로드된 이미지 간 순서는 마우스 드레그로 변경 가능
→ 순서 확인 후 업로드된 이미지로 일괄 저장 하기 누르면 새로 업로드한 이미지 순서대로 업로드 됨

10

[판매 시작] 눌러 온세일 하기

11

상품 온세일 후 수수료 등록을 위하여 “[국내T&A] 맵핑변경, 신규 상품 등록” 시트 내 신규 등록 내역 기재 (A-F열)
