## ngrok 설치 및 실행 (3분)

### 1단계: 다운로드 (1분)
브라우저에서:
https://ngrok.com/download

"Download for Windows" 클릭


### 2단계: 압축 풀기 (30초)
다운받은 ngrok.zip 압축 풀기
ngrok.exe를 현재 폴더에 복사


### 3단계: 회원가입 (1분)
https://dashboard.ngrok.com/signup
GitHub 계정으로 가입 (추천)


### 4단계: 인증 (30초)
로그인하면 자동으로 토큰 페이지 나옴
"Copy" 버튼 클릭

터미널에서 실행:
ngrok config add-authtoken YOUR_TOKEN


### 5단계: 실행! (10초)
터미널에서:
ngrok http 5000

생성된 URL 복사:
https://abc123.ngrok-free.app
