## 🚨 공통 반려 사유

1. **자동 수수료 공제 불가**: 수수료가 부과되는 상품의 자동 환불 기능 미개발
2. **취소 환불 연동 불가**: 마리트에서는 취소/환불 처리되었으나 공급사로 취소 불가
3. **미사용 시 환불 가능 규정**: 미사용 여부를 시스템적으로 자동 필터링 불가
4. **유효기간 내 환불 가능 규정**: 각 예약 건당 유효기간을 시스템적으로 자동 필터링 불가
5. **기타 이슈 발생**

## 📌 처리 프로세스 요약

```
반려 알럿 수신 
  → 반려 시트 확인 
    → 공급사 확인 
      → 해당 공급사 프로세스맵 참조 
        → 확인 수단으로 상태 조회 
          → 소통 수단으로 처리 요청 (필요 시)
            → 3.0 매니저 최종 처리
```
