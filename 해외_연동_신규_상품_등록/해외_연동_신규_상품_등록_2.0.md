## 🎯 등록 기준

- **캘린더 3개월 이상 오픈** 여부 
  - 단, 특별한 상황의 경우, 3개월 이내로 오픈 되어있어도 등록 가능
- **영어 or 한국어 지원** 여부 
  - 둘 중 하나의 언어라도 지원해야 검수 가능
- **즉시 확정 상품 우선 등록**
  - ※ 필요 시, 비즉시 확정 상품 보완용 허용

## 📊 상품 등록 방법

### 1단계: 생텀(Sanctum) — 공급사 상품 가져오기 + MRT ID 생성

**작업 순서:**

1. 생텀 → 좌측 메뉴 > 공급사 선택
2. 오른쪽 하단 파란 말줄임표 → 초록 가방 아이콘 **[공급사에서 상품 가져오기]** 클릭
3. 공급사 원본 ID 입력 후 등록
4. 팝업창에서 **[생성]** 클릭 → Offer ID 생성
5. 팝업 확인 후 자동 매핑 **[선택 적용]** : 공급사 > 생텀 까지 전달된 상품 페이지 정보를 2.0으로 적용

**선택적용 목록:** 어떤 항목이 적용 되는지 확인 가능

**📌 중요:**
- GYG, VIATOR, TIQETS, LINKTIVITY, BMG 는 **전체적용** 후 검수 시작 (대표사진 자동적용)
- 🔑 [상품관리 교육자료](https://docs.google.com/presentation/d/1o1OaWjbPQN3ks51SxGQ12KDFtYprDPLsDFyv5hRR8rg/edit?slide=id.g2c8a6d7a29c_1_74#slide=id.g2c8a6d7a29c_1_74) 81page 참고

### 2단계: 2.0 매니저 검수 시작 - 도시 선택 / 정보 수정

**2.0 매니저 페이지 접속하기:**
- 주소 창에서 오퍼 번호 자리에 생성한 **OFFER ID** 입력
- 예시: 102193

**도시 선택:**
- 도시 리스트 및 대표 도시 설정

**정보 수정:**

| 항목 | 설정 방법 |
|------|-----------|
| **표준 카테고리 (필수)** | [표준 카테고리 기준 표](https://docs.google.com/spreadsheets/d/1e048A8pk1SrXAkj4N-58Oe_9vQc1gYZoOr-aXo-gmrs/edit?pli=1&gid=2054470221#gid=2054470221) 참고하여 선택 |
| **1·2차 노출 카테고리** | [도시별 노출 카테고리 기준 표](https://docs.google.com/spreadsheets/d/1FjqymMxV8VPM5EWvUX9HkQZuFVkt-Ecz66vT9YJfxec/edit?gid=301657446#gid=301657446) 참고하여 선택 |
| **예약 가능 시간** | `0` (즉시 확정) |
| **예약 패스** | ✅ 확정 대기 자동 패스 (여행자 사이드 확정 대기 노출 안됨) |
| **취소/환불 가능 여부** | ✅ 여행자 사이드 취소 요청 버튼 노출 됨<br/>※ 취/환불 불가 상품은 V 해제 필요 |
| **상품 상세 페이지에서 옵션 노출** | 유효기간 내 사용하는 상품의 경우 ✅ |
| **상품 만료 기한 & 구매일로 부터 N일** | 유효기간 기재 |

**→ [적용] 버튼 클릭하여 저장**

### 3단계: 기본 정보 등록 - 어드민에서 상품 수정하기 1

**공지 사항 등록:**
- 부가 정보 옆 **연필모양** 버튼 클릭

**상품 상세 페이지 수정/등록 시작:**
- 기본정보 옆 **수정** 버튼 클릭

### 4단계: 기본 정보 등록 - 어드민에서 상품 수정하기 2

**※ [해외연동 상품 검수 템플릿 참고](https://docs.google.com/spreadsheets/d/1WE4yB_-9s0ZKDSB0-4EYkhuoeV97AnUJNzgj8WOmhAE/edit?gid=1004454496#gid=1004454496)**

**작성 항목:**

#### ★ 상품 이름
- 명확하고 간결하게 작성

#### ★ 한 줄 요약
- 상품의 핵심과 특징을 한 줄로 간단하게 요약

#### 여행 소개
**[이 티켓/투어/액티비티의 매력포인트!]**
- 상품의 셀링 포인트 세 가지 정도 기재
- 공급사마다 활용할 수 있는 bullet point 번역

#### 여행 상세
**[옵션 안내]:**
- 옵션명만으로는 각 옵션의 차이를 알기 힘들때 사용

**만나는 시간:**
- 만나는 시간이 딱 하나 명확한 경우 기재

**만나는 시간 추가 정보:**
- 픽업 시 만나는 시간 조율이 필요한 경우
- 만나는 시간이 하나 이상이고 선택할 경우
- 옵션마다 만나는 시간이 상이한 경우
- 픽업 장소에 따라 만나는 시간이 상이한 경우
- 문구는 템플릿 활용 바람

---

### 5단계: 기본 정보 등록 - 어드민에서 상품 수정하기 3

**이동 수단:**
- 차량 이동 / 도보 이동 / 그 외 중 선택
- (투어/액티비티/픽업 상품만 선택)

#### ★ 소요 시간
| 기간 | 설정 방법 |
|------|-----------|
| 30분 | minute 설정 후 30 기재 |
| 1시간 | hour 설정 후 1 기재 |
| 1일 | day 설정 후 1 기재 |
| 소요 시간 노출을 원하지 않는 경우 | flexible 설정 후 0 기재 |

**⚠️ 빈칸으로 비워두면 저장이 되지 않으니 유의**

**여행 규모:**
- 일행끼리 참여하는 프라이빗 상품의 경우에만 체크 박스 설정
- 디폴트값으로 체크되어 오픈됨 - 체크 해제 필요

---

### 6단계: 기본 정보 등록 - 어드민에서 상품 수정하기 4

**포함 사항:**
- 가격에 포함된 사항 입력
- 옵션별 포함 사항이 여행 소개 내 [옵션 설명]에 기재가 되어 있다면,
- ※ "옵션 별 포함 사항은 [ 옵션 안내 ]란을 확인해 주세요." 문구 작성

**불포함 사항:**
- 가격에 포함되지 않은 사항이 있을 경우 입력

#### ★ 사용 방법
- 공급사별 템플릿과 상품별 사용 방법에 맞게 기재
- **※ [해외연동 상품 검수 템플릿 참고](https://docs.google.com/spreadsheets/d/1WE4yB_-9s0ZKDSB0-4EYkhuoeV97AnUJNzgj8WOmhAE/edit?gid=1004454496#gid=1004454496)**

**티켓 사용 방법 명시:**

| 티켓 유형 | 설명 |
|----------|------|
| **e-티켓** | 모바일로 제시해 바로 입장 가능한 티켓<br/>e-티켓은 이용 방법이 무조건 바로 입장 |
| **실물 티켓** | 바우처 제시 후 실물 티켓으로 교환해야 하는 티켓<br/>실물 티켓은 입장 방법이 무조건 실물 티켓 교환 |

**투어 / 액티비티 카테고리:**
- 템플릿 공통 내용 & 상품 내 꼭 알아야 할 중요한 내용 명시
- 호텔 픽업의 경우, 호텔 픽업에 대한 공통 내용 추가 필요
- 미팅 장소 픽업 시, 만나는 장소 추가 안내 추가 필요

---

### 7단계: 기본 정보 등록 - 어드민에서 상품 수정하기 5

**태그 리스트:**
- 상품과 관련된 태그 설정 필요
- ※ 관련 태그 없으면 등록하지 않아도 상품 판매 가능

**만나는 장소:**
1. 업체 정보에 적힌 상품 주소를 구글지도에 검색
2. 만나는 장소 정보 (구글 지도 상 주소)
   - 예시: 파리 에펠탑 (Champ de Mars, 5 Av. Anatole France, 75007 Paris, 프랑스)
3. **※ "바우처 상의 만나는 장소를 꼭 확인 해 주세요" 필수 기재**

**위도/경도:**
- 구글 맵에서 빨간색 핀 우클릭 시 가장 상단에 위/경도 정보 복사

**체크박스:**

| 항목 | 설명 |
|------|------|
| 바우처 | 실물 티켓 교환 필요하다면 체크 |
| 해피콜 | 사용 X |
| 호텔 픽업 | 호텔 픽업 제공하는 상품 시 체크<br/>단 만나는 장소와 위도 경도 정보를 비워두어야 함<br/>(진행 x, 위도 경도 정보 반드시 기재 필요하며, 문구는 검수 템플릿 활용) |

**추가 정보:**
- 기재하지 않음

---

### 8단계: 기본 정보 등록 - 어드민에서 상품 수정하기 6

**취소/환불 규정 입력:**
1. 공급사 취소 환불 규정 꼼꼼히 확인 후 해당 내용에 맞는 자동 환불 템플릿 설정
2. 해당하는 자동 환불 템플릿 없는 경우 수기로 기재

**취소/환불 기준 일: 공급사 기준 + 1일로 설정**
- 예시: 공급사 : 2일 전 전액 환불 → MRT : 3일 전 전액 환불

#### ★ 노출 가격 / 노출 통화
- 생텀에서 선택 적용 눌렀다면 자동으로 대표가+통화 기준 따라옴

---

### 9단계: 기본 정보 등록 - 어드민에서 상품 수정하기 7

#### ★ 여행 사진

**업로드 방법:**
1. 여행사진 추가 > 파일선택 > 업로드
2. ※ position 숫자 : 사진 노출 순서 지정 가능

**대표 사진 정책:**
- 여행자 페이지에 노출되는 상품 대표 사진 정책상 **글자 없는 이미지만 가능**
- 따로 전달받은 기본 이미지가 없는 경우: 해당 상품 상세 페이지를 캡쳐해서 사용

**※ 최소한 4장의 대표사진을 등록해야 온세일 가능**

### 11단계: 판매 시작 (온세일)

**최종 확인 체크리스트:**
1. 앞서 진행한 상품 등록 과정 중 빠진 것이 없는지 꼼꼼히 확인
2. 특히 **취소 환불 규정** 반드시 확인할 것!
3. 검수가 끝났다면 **"판매시작"** 버튼 클릭
4. 여행자 페이지에서 **실제 노출/결제 테스트** 진행
