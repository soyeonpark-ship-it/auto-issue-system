⭕자동화툴 사용 방법

다운로드 링크

  • AICX MANAGER v0.3.3 (macOS)

  • AICX MANAGER v0.3.3 (Windows x64)

항목

상세 페이지

상세 내용

검색

  • 상품 id를 검색하여 조회

  • 우선검수 체크 박스 통해 우선 검수만 보기 및 검색 가능

  • 나에게 할당된 상품 체크 박스 통해 나에게 할당된 상품만 보기 및 검색 가능

  • 우측 드롭다운 통해 심사중, 판매중, 판매대기, 반려 상태값 설정하여 보기 및 검색 가능

신규 검수 / 재검수

  • 반려 처리한 이력이 있는 상품들은 재검수로 표시

  • !!! 주의 : 자동화툴을 이용해서 반려처리한 상품들만 재검수 표시 가능

    • 따라서 자동화툴 개발 이전의 상품들은 반려 이력이 있더라도 모두 신규 검수로 표시됨

상세 페이지

  • 검수 시작 : AI 검수 시작

  • 판매 시작 : AI 검수 후 반려 사항 없을 시 클릭

  • 심사 반려 : AI 검수 후 반려 사항 있을 시 클릭

  • 판매 대기 : 간혹 마이그레이션 상품 검수 시 사용

  • 사업실 검수 요청 : 사업실 검수 상품일 경우 클릭

  • 확인요청 (체크박스) : 엣지케이스 등 팀내 혹은 사업실 등과 확인이 필요한 경우 체크 - 검수 시간 카운팅에서 배제시킴

  • 이전 반려사유 확인하기 : 재검수 상품은 이전반려 사유 확인 가능

  • 파트너 ID & 상품 ID : 매니저페이지 링크가 심어져 있어 클릭 후 추가 서류 등 확인 가능

반려 케이스

  • 최종 결과 : 우측에 심사반려로 표시됨

  • 삭제 및 추가 : 반려 사유를 삭제하거나 수정하고 싶을 때 삭제 및 수정 버튼을 클릭하여 시행

  • 반려사유 추가 : 반려 사항에 대해 자동화툴이 누락한 사항의 경우 반려사유 추가 버튼을 클릭하여 추가

수수료 확인 및 수정

  • 판매 수수료율 : T&A 에 책정되어 있는 기본 수수료 (20%) 를 확인할 수 있음

  • 별도 수수료 정책 확인 : 예외적으로 다른 수수료가 반영되는 파트너들 리스트를 확인할 수 있음

  • 수수료 수정하기 : 수수료 수정이 필요할 시 매니저페이지로 이동가능하도록 되어 있음

⭕검수 가이드라인

  • 매뉴얼 슬라이드

  • 검수 가이드라인 시트

  • 동의서 응답시트
